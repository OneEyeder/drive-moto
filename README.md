# Jet Ski Shop Website DRIVE MOTO

Welcome! This project showcases a modern, responsive online store built with HTML, CSS, Sass, JavaScript, and Swiper.js.

---

## Features

- **Beautifully Designed UI:** Crafted with Sass for modularity and maintainability.
- **Responsive Layout:** Ensures seamless experience on desktops, tablets, and smartphones.
- **Interactive Slider:** Powered by Swiper.js for showcasing featured Jet Skis and promotions.
- **Easy Navigation:** Clear menu structure and smooth scrolling.
- **Modern Technologies:** Utilizes HTML5, CSS3, Sass, JavaScript, and Swiper.js.

---

## Technologies Used

- **HTML5** – Structure of the website
- **CSS3 & Sass** – Styling and theming
- **JavaScript** – Interactivity and dynamic content
- **Swiper.js** – Responsive slider component
